# Vite-Electron

This is the source of the Dev.to tutorial in:
https://dev.to/rafaelpimpa/how-to-create-electron-app-with-vite-2-with-minimal-boilerplate-4did

## Install dependencies

```
npm install
```

## Run dev server

```
npm run electron:dev
```

## Build app

```
npm run electron:build
```
